# `_templates/` — Templates institucionais

Modelos de documento corporativos (Word, Excel, etc.) usados para preparar entregáveis no padrão visual da Coreon.

- `Padrão Coreon Word 1.docx` — template Word com a identidade visual da Coreon, usado para documentos formais (specs externas, relatórios, atas).

Não é input de feature específica (esses ficam em `specs/features/_inputs/<family>/`). É um asset compartilhado pelo workspace.
